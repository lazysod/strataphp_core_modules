# OAuth Clients Module

This module provides a framework for integrating OAuth and SSO similar to providers like (Google, Facebook, etc.) into StrataPHP. Use as a hub or installable extra.

## Features
- Modular provider support
- Easy integration with user/admin modules
- Configurable via `config.php`

## Usage
Add provider classes to `Providers/` and update `config.php` with keys/secrets.
